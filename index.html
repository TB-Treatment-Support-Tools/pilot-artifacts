TB Treatment Support Tools Pilot Study

The TB Treatment Support Tools (TB-TSTs) are a set of software tools that help monitor adherence to TB medication regmines and provide support to patients throughout treatment. This site describes an earlier set of the tools used in 2019 pilot study in Buenos Aires, Argentina.
